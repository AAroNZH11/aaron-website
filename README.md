# Aaron Zhong — Personal Website

Personal portfolio website for Aaron Zhong, Graduate Student in Biostatistics at the University of Washington.

## Live Site

[https://aaronznh11.github.io/aaron-website/](https://aaronznh11.github.io/aaron-website/)

## Sections

- **About** — Bio, background, and social links
- **Experiences** — Work and teaching experience
- **Projects** — Selected data science and research projects
- **Contact** — Email and footer

## Tech Stack

- Static HTML/CSS/JavaScript
- Bootstrap 3
- jQuery + plugins (Stellar, ScrollReveal, Owl Carousel, Isotope)
- Hosted via GitHub Pages

## Local Development

Open `index.html` directly in a browser, or run a local server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
